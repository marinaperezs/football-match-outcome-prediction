# ⚽ Football Match Outcome Prediction

An end-to-end machine learning project focused on predicting football match outcomes using historical match data.

The objective of this project is not only to build a predictive model, but also to develop a complete data science workflow, including data collection, storage, preprocessing, feature engineering, model training and evaluation.

---

## 📌 Project Overview

This project automatically retrieves historical football match data through the Football-Data.org API, stores it in a local SQLite database and uses Python and Scikit-learn to build predictive machine learning models.

The project is currently under active development, with the main focus on improving feature engineering and predictive performance.

---

## 🚀 Current Features

- Automated data collection using the Football-Data.org API
- SQLite database for historical match storage
- Data preprocessing using Pandas
- Initial Random Forest classification model
- Match outcome prediction pipeline
- Model evaluation
